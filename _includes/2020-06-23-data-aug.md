Consider the set of augmentations applied to a dataset. These are functions from the data domain to itself. For example adding random noise is such a function
$$f: X \to X, f(x) = x + \mathcal{N}(0, \sigma)$$.

If we lift it up to the space of functions, with the main operator being composition, then $$(f \circ f)(x) = x + \mathcal{N}(0, 2\sigma)$$ by convolution.
